Firebase Storage Tutorial

Learn how to connect Firebase Storage to your web application for file uploads

1 Create a Firebase Project

1 Go to Firebase Console

Visit Firebase Console and sign in with your Google account.

2 Create a New Project

  1. Click on "Add project" or "Create a project"
  2. Enter a project name (e.g., "my-file-uploader")
  3. Follow the setup steps (Google Analytics is optional)
  4. Click "Create project" and wait for it to initialize

2 Set Up Firebase Storage

1 Enable Storage

  1. In your Firebase project dashboard, click on "Storage" in the left menu
  2. Click "Get started"
  3. Choose your security rules (start in test mode for development)
  4. Select a location for your storage bucket
  5. Click "Done"

2 Get Configuration Details

  1. Go to Project Settings (click the gear icon next to "Project Overview")
  2. Scroll down to "Your apps" section
  3. Click the web icon to add a new web app
  4. Register your app with a name (e.g., "File Uploader")
  5. Copy the configuration object

3 Integrate Firebase with Your Code

1 Add Firebase SDK

Include the Firebase SDK in your HTML file by adding these scripts to the head section:

<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-app-compat.js"></script>
<script src="https://www.gstatic.com/firebasejs/9.22.0/firebase-storage-compat.js"></script>

2 Initialize Firebase

Add this code to your JavaScript, replacing with your actual Firebase config values:

// Your web app's Firebase configuration
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "123456789",
  appId: "YOUR_APP_ID"
};

// Initialize Firebase
firebase.initializeApp(firebaseConfig);
// Get a reference to the storage service
const storage = firebase.storage();

4 Implement File Upload Function

1 Create Upload Function

Add this function to handle file uploads to Firebase Storage:

function uploadFileToFirebase(file) {
  // Create a storage reference
  const storageRef = firebase.storage().ref();
  const fileRef = storageRef.child('uploads/' + file.name);

  // Upload file to Firebase
  const uploadTask = fileRef.put(file);

  // Monitor upload progress
  uploadTask.on('state_changed',
    (snapshot) => {
      // Progress monitoring code here
    },
    (error) => {
      // Handle errors
    },
    () => {
      // Upload completed successfully
      uploadTask.snapshot.ref.getDownloadURL().then((downloadURL) => {
        // Get download URL and update UI
      });
    }
  );
}

5 Deploy Your Application

1 Choose a Hosting Platform

You can deploy your application to:

  • Firebase Hosting - Directly integrated with Firebase
  • Netlify - Drag and drop deployment
  • Vercel - Connect your GitHub repository
  • GitHub Pages - Free hosting for static sites

2 Configure Storage Rules

For production, update your Firebase Storage rules to secure your data:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /uploads/{allPaths=**} {
      allow read, write: if request.auth != null;
      // Or customize based on your needs
    }
  }
}
