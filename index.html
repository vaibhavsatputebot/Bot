How to actually upload to GitHub

This demo simulates GitHub file uploads. To actually upload files to GitHub:

  1. Create a repository on GitHub
  2. Use GitHub's web interface to upload files
  3. Or use Git commands: 
    git add .
git commit -m "Add files"
git push
  4. For automated uploads, use the GitHub API
