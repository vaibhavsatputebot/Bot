How to upload files to GitHub

This demo simulates GitHub file uploads. To actually upload files to GitHub:

  1. Go to your repository on GitHub
  2. Click "Add file" → "Upload files"
  3. Drag and drop or select files to upload
  4. Add a commit message and commit changes
  5. For automated uploads, use the GitHub API with a personal access token
